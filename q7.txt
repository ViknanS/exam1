3. GITHUB ACTIONS WORKFLOW

---------

File Path:
.github/workflows/docker.yml

## Code:

name: Docker CI Pipeline

on:
push:
branches: ["main"]

jobs:
build:
runs-on: ubuntu-latest

```
steps:
- name: Checkout Code
  uses: actions/checkout@v3

- name: Login to Docker Hub
  uses: docker/login-action@v2
  with:
    username: ${{ secrets.DOCKER_USERNAME }}
    password: ${{ secrets.DOCKER_TOKEN }}

- name: Build Docker Image
  run: docker build -t ${{ secrets.DOCKER_USERNAME }}/myapp .

- name: Push Docker Image
  run: docker push ${{ secrets.DOCKER_USERNAME }}/myapp
```

-------------

4. CREATE DOCKER HUB ACCESS TOKEN

---

Steps:

1. Login to Docker Hub
2. Go to Account Settings
3. Click Security
4. Click "New Access Token"
5. Name: github-actions
6. Copy the generated token

NOTE:

* Do NOT use your password
* Always use access token for security

---

5. CREATE NEW GITHUB REPOSITORY

---

Steps:

1. Go to GitHub
2. Click "New Repository"
3. Name: exam1 (or any name)
4. Do NOT initialize with README
5. Click Create

6. ADD SECRETS IN GITHUB

---

Steps:

1. Open your GitHub repository
2. Go to Settings
3. Click "Secrets and variables" → Actions
4. Click "New repository secret"

Add the following:

DOCKER_USERNAME = your_dockerhub_username
DOCKER_TOKEN = your_access_token
